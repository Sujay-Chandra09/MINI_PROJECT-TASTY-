# miniproject
🍽️ Tasty – Smart Recipe & Meal Planner App

A full-stack MERN-based recipe application that helps users discover recipes, plan meals, and manage food habits efficiently with personalized recommendations.

🚀 Overview

Tasty is a modern full-stack web application built using the MERN stack (MongoDB, Express, React, Node.js) that allows users to explore recipes, manage favorites, plan meals, and generate smart suggestions.

👉 A good README should clearly explain what the project does, how it works, and how to use it

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/CNU2005/MINI_PROJECT-TASTY-.git
cd MINI_PROJECT-TASTY-
2️⃣ Setup Backend
cd TastyBackend
npm install
npm start
3️⃣ Setup Frontend
cd TastyFrontend
npm install
npm run dev
🔑 Environment Variables

Create .env file in backend:

PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
